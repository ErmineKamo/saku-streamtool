This site is meant to be used as an OBS widget.
Please copy the link from your address bar and add it to a Browser Source.
Use the width/height of x, and the default custom CSS value.
Then, right-click the Browser Source and use the Interact button to work with the app. The script playback will begin after a left-click in the Interact window.
If you would like to bypass this check and run the app in your normal browser, add &force to the end of the URL in your address bar.
